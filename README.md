# Friendly 5

A multiplayer card game built with React, TypeScript, Vite, Express, and Socket.IO.

## Run locally

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`

## Deploy on Render

Render can deploy this project as a Node web service.

Use these settings:
- Build Command: `npm install && npm run build`
- Start Command: `npm start`

A Render blueprint is included in [render.yaml](render.yaml).
